Seuls les détenteurs d'une **clé API valide** fournis préalablement par Tulip peuvent être utilisées.
Lors d'un appel API, vous devez indiquer cette __clé dans l'en-tête__ de la requête avec la dénomination ***key***

```txt
key=Ok3DiewO2ID4jkdsa
```
